# Daily report

1. Ask for or infer only the work items, outcomes, blockers, and next steps explicitly available in the conversation.
2. Match content to the visible daily-report fields on the current page.
3. Keep the report concise, factual, and easy for the user to edit.
4. Fill a draft and verify every changed field.
5. Never submit or publish the report without a separate explicit confirmation.
