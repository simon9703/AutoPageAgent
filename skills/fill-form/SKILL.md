# Fill form

1. Match requested values only to visible interactive elements from the current snapshot.
2. Prefer labels, roles and placeholders over element order.
3. Fill only values explicitly supplied or approved by the user.
4. Do not submit, publish, send, pay, delete, or approve unless separately and explicitly requested.
5. Always return a confirmation-required action plan.
