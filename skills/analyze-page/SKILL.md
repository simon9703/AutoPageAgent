# Analyze page

1. Use only evidence contained in the current page snapshot.
2. Explain the page purpose, major sections, important fields, and visible interaction points.
3. For performance questions, cite navigation timing and concrete slow or large resources.
4. Separate observed evidence from inference.
5. Never produce browser actions unless the user explicitly asks to change or operate the page.
